Exercise page

    Let’s start the party! 🥳

    • 1.) Use CarService to communicate with API and get first preconfigured cars data. We have already prepared an test API call GET /car, you can see tha data in console after visiting of Exercise page.

    • 2.) Find app/models/car.vo.ts file and prepare your Car model depends on parameters, that you will get from http://localhost:3000/car response:

      • id: number
      • name: string
      • …
      • …

    • 3.) Create CarCardComponent – this component will be used to show short car information on our “Cars” page.

      Content that must be shown in car card:

      • Model - model of car
      • FirstRegistration - registration date
      • Delete button – used to remove our car
      • Details button – navigate to car details page, where will be displayed all car information

    • 4.) Display preconfigured cars with help of CarCardComponent on our “Cars” page.

    • 5.) Create new Component, that will be used to create new cars and bind it to new navigation link (“Add car”)

      Create car form:

      • Name field is required
      • FirstRegistration field is required
      • Origin field is required and must be of type "GERMANY" or "USA"
      • Save button must be active if all fields are valid

    • 6.) Consume POST endpoint to create new car.

    • 7.) Create CarDetailsComponent to show car information. For navigation must be used our “Details” button from car card.

    • 8.) Consume DELETE endpoint to remove car from “Cars” page at click on “Delete” button of car card.

    Already finished? Try next bonus steps!

    • 1.) Write the unit test of your choice.

    • 2.) Write an interceptor and display in console tab every request.

    • 3.) Try to render car details page only after receiving data from API (use angular Resolve interface). Resolve interface helps us to wait data, that should be resolved before route is finally activated

    • 4.) Style your website.

    • 5.) Consume GET filter endpoint to filter our cars on “Cars” page.

    • 6.) You are unreal! Take a coffee =)
